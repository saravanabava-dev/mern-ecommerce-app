# MERN E-Commerce Laundry Application

A full stack MERN application for online laundry and ironing service booking. Users can browse services, place orders, manage profiles, and make payments through an interactive web interface.

---

## 🚀 Features

* User authentication and authorization
* Google OAuth login suppor
* Laundry and ironing service booking
* Product and service management
* Order management system
* Razorpay payment integration
* Image upload support using Multer
* Interactive frontend with React
* Responsive UI design
* Location and map integration
* Dashboard and analytics charts

---

## 🛠️ Tech Stack

### Frontend

* React.js

* Recharts

### Backend

* Node.js

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/saravanabava-dev/mern-ecommerce-app.git
```

---

## ▶️ Backend Setup

```bash
cd Backend
npm install
npm run start:dev
```

---

## ▶️ Frontend Setup

```bash
cd Frontend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the `Backend` folder.

Example:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
RAZORPAY_KEY=your_key
RAZORPAY_SECRET=your_secret
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
```
